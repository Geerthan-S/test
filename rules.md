# Frontend Quality Standards

## Layout Rules

- Use a maximum content width of 1280px.
- All sections must use consistent horizontal padding.
- Maintain an 8px spacing system.
- Section vertical spacing must be 80px-120px.
- Never place elements without checking alignment against surrounding content.

## Alignment Rules

- All headings must align with section content.
- Cards in the same row must have equal heights.
- Buttons must align consistently across sections.
- Images and text blocks must follow a common grid.
- No element should appear visually off-center.

## Responsive Rules

- Test desktop, tablet, and mobile before completing any task.
- No horizontal scrolling.
- Typography must scale appropriately.
- Images must remain proportional.

## Design Consistency

- Reuse existing colors, shadows, borders, and spacing.
- Do not introduce new styles unless requested.
- Maintain a premium corporate appearance.

## Before Finishing Any Task

Perform a UI audit:

1. Check alignment.
2. Check spacing consistency.
3. Check responsiveness.
4. Check visual hierarchy.
5. Check accessibility.
6. Check hover and interaction states.

Do not mark a task complete until all checks pass.

## Specialist Roles

### 1. Frontend QA Auditor (Highest Priority)

Runs after every change.

Act as a senior frontend QA engineer.

After every UI change:

- Audit alignment.
- Audit spacing consistency.
- Audit responsiveness.
- Audit typography hierarchy.
- Audit button consistency.
- Audit image sizing.
- Audit section spacing.

Do not stop at implementation. Review the page and suggest improvements until it meets production-quality standards.

Treat every task as incomplete until a full UI audit passes.

### 2. Design System Enforcer

Prevents random styling.

Act as a design system guardian.

Before adding any new UI element:

- Check existing design patterns.
- Reuse existing spacing.
- Reuse existing colors.
- Reuse existing typography.
- Reuse existing component styles.

Do not create new visual patterns unless explicitly requested.

Maintain a consistent premium corporate design language across the entire website.

### 3. Responsive Layout Specialist

Act as a responsive layout expert.

For every page:

- Verify mobile layout.
- Verify tablet layout.
- Verify desktop layout.
- Prevent horizontal scrolling.
- Maintain proper spacing at all breakpoints.
- Ensure text remains readable.
- Ensure images scale correctly.

Test responsiveness before completing any task.

### 4. Hero Section Specialist

Since most construction websites live or die by their hero section.

Act as a premium landing page designer.

For every hero section:

- Create strong visual hierarchy.
- Ensure CTA visibility.
- Ensure proper content positioning.
- Maintain premium spacing.
- Use modern corporate design patterns.
- Optimize for first impressions.

The hero section should immediately communicate trust, professionalism, and scale.

### 5. Pixel Perfect Alignment Specialist

Act as a pixel-perfect frontend engineer.

Review every section for:

- Vertical alignment.
- Horizontal alignment.
- Container consistency.
- Grid consistency.
- Card consistency.
- Equal spacing.

Identify even minor visual misalignments and correct them before considering the task complete.
